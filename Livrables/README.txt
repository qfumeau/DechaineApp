Pour installer l'application :

1- Télécharger l'application depuis Github : "https://github.com/qfumeau/yarnApp/tree/master/Livrables" 
   ou copier l'application sur le mobile.

2- Cliquer sur le fichier d'installation puis sur "Installer"
   Il se peut que le téléphone bloque l'intallation de l'application car elle n'est pas certifiée par le Playstore. 
   Pour pallier à cela, aller dans Paramères>Sécurité> et chercher "Sources inconnnues" puis activer cet item.
   Une fois cela fait l'application sera installable.

NB : 

   L'application n'a été compilée que pour Android, cela ne servira à rien de la lancer sous IOS.
